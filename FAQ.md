# FAQ

## Do I need to install anything?

You only need Python. The generator does not need extra packages.

## Which option should I choose?

Choose `Strong password` for most accounts. Choose `Memorable passphrase` when you need to type or remember it.

## Is the password saved anywhere?

No. The app prints the password on your screen and optionally copies it to your clipboard.

## Can I use this for real accounts?

Yes, but store the password in a password manager and use a different password for each account.
