# Examples

Open the friendly menu:

```bash
python password_generator.py
```

Generate a readable password:

```bash
python password_generator.py --preset basic
```

Generate a strong password:

```bash
python password_generator.py --preset strong
```

Generate a memorable password without symbols:

```bash
python password_generator.py --preset memorable
```

Generate a passphrase:

```bash
python password_generator.py --passphrase --words 5 --capitalize --add-number
```
